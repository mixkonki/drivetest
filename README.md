# Drive Test - Διαχείριση Προφίλ Χρήστη

## Περιγραφή Έργου
Μια ολοκληρωμένη εφαρμογή διαχείρισης προφίλ χρηστών με δυνατότητες:
- Επεξεργασίας προσωπικών στοιχείων
- Ενημέρωσης διεύθυνσης 
- Προβολής τοποθεσίας σε χάρτη
- Ανεβάσματος avatar

## Τεχνολογίες
- PHP
- JavaScript
- MySQL
- Google Maps API
- jQuery

## Απαιτήσεις
- PHP 7.4+
- MySQL
- Web Server (Apache/NGINX)
- Σύνδεση στο Internet

## Εγκατάσταση
1. Κλωνοποιήστε το repository
2. Ρυθμίστε τη βάση δεδομένων
3. Προσθέστε το Google Maps API Key
4. Ρυθμίστε τις παραμέτρους σύνδεσης
